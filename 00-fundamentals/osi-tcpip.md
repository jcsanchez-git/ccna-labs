# OSI vs TCP/IP

## Modelo OSI
1. Capa 1 — Física
2. Capa 2 — Enlace de datos
3. Capa 3 — Red
4. Capa 4 — Transporte
5. Capa 5 — Sesión
6. Capa 6 — Presentación
7. Capa 7 — Aplicación

## Modelo TCP/IP
1. Capa de enlace
2. Capa de red
3. Capa de transporte
4. Capa de aplicación

## Mapeo rápido OSI ↔ TCP/IP
OSI 7 → TCP/IP 4
- Aplicación (7,6,5) → TCP/IP Aplicación
- Transporte (4) → TCP/IP Transporte
- Red (3) → TCP/IP Red
- Enlace (2,1) → TCP/IP Enlace
