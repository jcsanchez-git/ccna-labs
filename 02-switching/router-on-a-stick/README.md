# Router-on-a-Stick (Inter-VLAN Routing)

Este módulo documenta el concepto de Router-on-a-Stick (RoAS),
una técnica de interconexión entre VLANs utilizando un único
enlace físico configurado como trunk.

El objetivo es comprender:
- Cómo funciona el enrutamiento entre VLANs
- El rol de las subinterfaces en el router
- La relación entre VLANs, trunking y routing
- En qué escenarios se utiliza Router-on-a-Stick

Este módulo es conceptual y no incluye configuraciones.
