# VLAN Hopping

Este módulo documenta el concepto de VLAN Hopping, una vulnerabilidad
de Capa 2 asociada a configuraciones inseguras de switches.

El objetivo es comprender:
- Qué es VLAN Hopping
- En qué condiciones ocurre
- Por qué no es un ataque “mágico”
- Cómo se mitiga mediante buenas prácticas

Este módulo es conceptual y no incluye configuraciones.
